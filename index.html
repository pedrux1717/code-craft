// Capabilities

Cosa fa AIROP

Un sistema distribuito che raccoglie metriche, le analizza con ML, interviene autonomamente e mostra tutto in real-time.

⬡

Raccolta Metriche Distribuita

Gli Agent Python raccolgono ogni 5 secondi CPU, RAM, disco, rete e processi da ogni host della rete. Supporto multi-agent con configurazione YAML e retry automatico. Ogni agent invia heartbeat periodici per segnalare il proprio stato.

psutilYAML configHeartbeatRetry logic
◎

AI Anomaly Detection

L'AI Engine analizza le metriche con Isolation Forest per rilevare anomalie e Linear Regression per prevedere trend. Genera alert automatici su CPU, RAM e disco prima che il problema diventi critico.

Isolation ForestLinear Regressionsklearn
↺

Self-Healing Automatico

L'Automation Engine applica regole configurabili: kill process, restart service, scale up/down, clear cache. In caso di anomalia non risolvibile, escalation automatica all'amministratore con tracking completo di tutte le azioni.

Kill ProcessRestart ServiceScale Up/Down
▦

Dashboard Real-Time

Interfaccia React con aggiornamento automatico ogni 10 secondi. Grafici interattivi per CPU e RAM, distribuzione alert per severità, stato agenti connessi e storico azioni di automazione.
